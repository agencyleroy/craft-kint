# craft-kint
Craft CMS plugin for Kint debugging

## Usage

```twig
{{ d(craft) }}
{{ d() }}
```

## Installation with composer

```json
"repositories": [
  {
    "type": "vcs",
    "url": "https://github.com/agencyleroy/craft-kint"
  }
]
```

```console
foo@bar:~$ composer require agencyleroy/craft-kint
```
